SOBRE MIM

Olá, eu sou a Greice! Programadora por formação (Técnica em Informática para Internet pelo IFRS) e por curiosidade incessante. O que mais me motiva na tecnologia é a sua abrangência – a possibilidade de sempre aprender algo novo e de transformar linhas de código em soluções reais.

Atualmente, navego com confiança pelo ecossistema web (HTML, CSS, JS, React) e estou expandindo meu conhecimento para o back-end com Java. Meu horizonte, no entanto, está nos dados: meu objetivo é evoluir da criação de sistemas para a análise inteligente da informação que eles geram.

Estou em busca da minha primeira experiência profissional como desenvolvedora, um passo crucial para consolidar minha base técnica e iniciar essa jornada rumo à análise de dados. Nos meus momentos offline, você me encontra levantando pesos, explorando universos cinematográficos ou focando em mirar bem em algum jogo de FPS.

PROJETOS

Ellie's Journey - Memory Game

HTML CSS JavaScript

"Ellie's Journey" é um jogo da memória imersivo que transporta o jogador para o universo sombrio de The Last of Us, onde o objetivo é encontrar pares de cartas com personagens e símbolos icônicos da série. Desenvolvido com HTML5, CSS3 e JavaScript, o projeto apresenta um design temático responsivo, animações fluidas no virar das cartas e uma lógica de jogo eficiente — incluindo embaralhamento aleatório, contagem de movimentos e feedback visual/sonoro —, consolidando conceitos fundamentais de front-end em uma experiência interativa e visualmente cativante.

Mega Sena - Simulator

HTML CSS JavaScript

"Sena Simulator" é um projeto interativo que replica a experiência de um sorteio da Mega-Sena, permitindo ao usuário escolher números aleatórios para suas apostas e simular o resultado de um concurso. Desenvolvido com HTML5, CSS3 e JavaScript, o jogo aplica lógica de programação para criar sorteios únicos (sem repetições), comparar os números apostados com os sorteados e calcular acertos, tudo apresentado em uma interface visual limpa e responsiva que simula a emoção do jogo real.

Ver Projeto GitHub

Snake Game

HTML CSS JavaScript

"Snake Game" é uma recriação interativa do clássico jogo da cobrinha, onde o jogador controla uma serpente que deve comer itens espalhados pela tela sem colidir com as paredes ou com o próprio corpo. Desenvolvido com HTML5, CSS3 e JavaScript, o projeto implementa lógica de movimentação contínua, detecção de colisões, aumento dinâmico do corpo da cobra e um sistema de pontuação em tempo real, demonstrando o uso do elemento Canvas para renderização gráfica e controle de estados de jogo.
